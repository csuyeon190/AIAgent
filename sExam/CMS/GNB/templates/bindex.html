Seamless한 사용성

Brity Copilot 4대 업무 영역별 AI 기능 인포그래픽. 중앙에 메일/결재·메신저·미팅·오피스 4개 아이콘이 원형으로 배치되고, 각 영역별 기능이 말풍선으로 연결됨. 메일/결재: 메일·결재 요약, 메일·결재 작성, 답변 메일 작성. 메신저: 정보 검색, 메시지 수정, 대화 내용 요약. 미팅: 실시간 자막·번역, 특정인 발언 요약, 회의록 생성. 오피스: 정보 검색·요약, 문서 포맷 변환, 초안 생성. 각 영역에는 Brity Copilot UI 스크린샷이 함께 표시됨.

메일, 메신저, 미팅, 오피스
내가 일하는 위치 어디에서나 코파일럿을 즉시 활용해 업무를 효율적이고 스마트하게 이어갈 수 있습니다.
브리티웍스의 어떤 서비스에서나 편리하게 생성형 AI 전용 탭에서 사용해보세요.

사용자 친화적 기능

Brity Copilot AI 글쓰기 및 사내시스템 검색 기능 UI 스크린샷 두 가지. 왼쪽: 메일 작성 화면에서 ‘AI 글쓰기’ 기능으로 회의목적과 일정을 입력하면 회의초대 메일을 자동 작성하는 모습. 업무요청·회의초대·진행상황 공유 등 추천 템플릿 버튼 표시. 오른쪽: Brity Copilot 채팅 패널에서 사용자가 ‘사내시스템에서 키워드 검색해줘’라고 요청하면 전체·드라이브·메일함·사내시스템 중 검색 범위를 선택하여 DX LLM 기반으로 답변하는 대화 UI.

생성형 AI 사용 경험이 적은 사용자도 원하는 답과 결과를 얻을 수 있도록, 손쉽게 사용이 가능한 다양한 추천 프롬프트와 칩 메뉴 형태의 UI를 제공합니다.

사내 업무 시스템 연계 지원

Brity Copilot 내부 데이터 연동 아키텍처 다이어그램. 왼쪽 상단 Brity Drive(K:)에 개인폴더·공유폴더·부서폴더 구조 표시. 상단 중앙 ‘내부 데이터’ 영역은 협업 솔루션(메일·메신저·드라이브 아이콘)과 업무 시스템(HR·재무/회계·ERP 등 큐브 아이콘)으로 구성. 메일·메시지·문서·데이터가 화살표를 통해 하단 Brity Copilot 엔진으로 흘러들어가며, Connector(Plug-in)와 Knowledge Search 두 레이어를 거쳐 LLM과 양방향으로 연결됨. 왼쪽 하단에 노트북으로 질문하는 직원 일러스트.

외부 검색으로 찾을 수 없는 내부 데이터를(메일, 메시지, 문서 및 기업의 업무 시스템 데이터) Plug-In 형태로 연계하여 코파일럿을 통해 편리하게 통합 검색이 가능합니다.

명확한 커뮤니케이션 지원

Brity Meeting 화상회의 실시간 자막 스크립트 기능 스크린샷. 4인 화상회의 화면에서 현재 발언 중인 참가자(김삼성)의 영어 발언이 화면 하단에 실시간 자막으로 표시됨. 오른쪽 ‘자막 스크립트’ 패널에는 한국어 표시 옵션과 함께 존 빌레·김선성 등 참가자별 발언이 영어 원문과 한국어 번역 텍스트로 시간순 나열됨. 하단 툴바에 마이크·카메라·공유·녹화·자막 스크립트·채팅·종료 버튼 표시

정확한 음성 인식률을 바탕으로 (한국어 정확도 94% 이상) 실시간 자막을 제공하며 코파일럿을 통한 회의 요약 및 특정인 발언 요약 등의 기능을 제공합니다.
회의 종료 후, 자동 회의록이 생성되며 참석자에게 공유가 가능합니다.

생성형 AI 보안 관리

Brity Copilot 보안 정책 인포그래픽. 상단에 자물쇠 방패 아이콘과 ‘Brity Copilot’ 로고. 왼쪽 파란색 영역 ‘질의 가능(자유로운 검색)’: Market Trend·최신 뉴스·공개 자료 등 외부 공개 정보는 검색 및 질의 허용. 오른쪽 분홍색 영역 ‘질의 불가(데이터 차단)’: 내부 경영정보·개인정보·소스 코드 등 민감 정보는 AI 외부 유출 차단(X 표시). 두 영역을 대비하여 보안 경계를 시각적으로 표현.

키워드 및 메시지 패턴 분석으로 내부 주요 정보나 소스코드 입력 시 자동으로 발송 차단하여 내부 정보를 안전하게 보호합니다.
또한 사용자 권한 관리 기능을 통해 개인별 내부 데이터 접근 권한 관리도 가능합니다.

    자주 묻는 질문

    • 브리티 코파일럿은 브리티웍스(협업솔루션)를 도입해야 사용이 가능합니다.
      브리티 코파일럿은 브리티웍스 환경에 최적화가 되어 있기 때문에, 기업에서 사용하고 계신 기존 협업 서비스에 add-in 형태로 제공하지는 않습니다.
      다만, 서비스별( 메신저 +코파일럿 / 미팅 + 코파일럿) 단독으로 도입은 가능합니다.

    • 서비스별로 단독 도입이 가능합니다.
      예) 메신저 +코파일럿 / 미팅 + 코파일럿 형태로 단독 도입은 가능합니다.
      단, 미팅에서 메신저 공유와 같이 서비스간 연계가 필요한 기능들은 제한될 수 있습니다. 자세한 사항은 여기를 클릭해 문의 부탁드립니다.

    • 브리티웍스와 코파일럿은 SaaS 서비스로 별도의 구축 없이 바로 사용이 가능합니다.
      도입 또는 상담이 필요하면 상담 문의하기를 통해 연락해 주세요.

    • Brity Copilot 활용 중 사용자가 입력한 프롬프트 및 메일/대화/회의록 요약 등에 사용되는 내용 등은 LLM 학습에 활용되지 않습니다.
